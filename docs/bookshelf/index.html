Machine Learning Teórico

Essential Math for AI
Hala Nelson
An Introduction to Statistical Learning
Trevor Hastie & Robert Tibshirani

Machine Learning

Machine Learning For Absolute Beginners
Oliver Theobald
Explica desde 0. No tiene muchos fundamentos matemáticos ni mucho código, es ideal si querés entender, muy por arriba, de qué se trata el Aprendizaje Automático y y cuáles son sus principales enfoques.
The Hundred-Page Machine Learning Book
Andriy Burkov
Está bueno si querés tener una visión general de la parte matemática de los modelos de ML, en cuanto al código el código quedo un poco obsoleto ya que el libro tiene sus años, hay que hacer unas modificaciones y adaptaciones para que pueda funcionar.
Hands-On Machine Learning
Aurélien Géron
Grokking Machine Learning
Luis G. Serrano
